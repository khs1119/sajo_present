음성 대화형 진단Simple

촬영 외 모든 진단은 음성 설문으로
SOLUTION
  • 촬영을 뺀 생활습관 진단(수면·물·스트레스·운동)을 타이핑 없이 목소리로
    손이 바쁜 아침/취침 전에도 부담 없이 완료
  • AI가 질문을 읽어주고(TTS) 사용자가 말하면 자동 변환(STT) → AI와 대화하듯 양방향
① '번거로움'을 없앤다 · 세부 기능

자동 진단 스케줄Smart

매일 설정해둔 같은 시간에 자동 진단 ON
SOLUTION
  • 진단 시간을 한 번 설정하면 매일 같은 시간에 자동으로 켜짐 → 언제 할지 결정하는 부담 제거, 자연스러운 루틴화
  • 설정 시간에 자동으로 앱 on → 진단을 '기억하고 직접 실행해야 하는 일'에서 '알아서 챙겨주는 일'로 전환
① '번거로움'을 없앤다 · 세부 기능

데이터 기반 맞춤 제품 추천Smart

피부 진단 기반 맞춤형 제품을 매주 자동 추천
SOLUTION
  • 제품 탐색·비교 필요 X 촬영 데이터 + 음성 답변을 함께 분석해서 제품 추천 → 선택 피로 제거
  • 단순 추천이 아니라 데이터 근거 성분 + 이번주 사용 TIP까지 제공 → 왜 이 제품인지, 어떻게 쓰는지 자동 안내
  • 코스맥스/고객사 제조 제품 중 진단 수치 기반 매칭
① '번거로움'을 없앤다 · 세부 기능

자동 문앞 배송Speed

주간 추천 제품은 매주 정해진 요일에 자동 배송
SOLUTION
  • 진단만 하면 제품은 알아서 도착 → 주문·재구매 행동 자체를 제거
  • 진단 결과에 따라 사용량·교체 주기를 유동 조절(우선 1주 소량) → 낭비 없이 딱 필요한 만큼만
  • 친환경 용기 배송으로 구독형 소비의 환경 부담 완화

진단의 루틴화

진단 시간 설정 시 정해진 시간에 자동 앱 작동
SOLUTION
  • '진단 시간 변경'으로 아침 준비·나이트 루틴 등 내 생활 패턴에 맞게 언제든 재설정
  • 굳이 기억하거나 앱을 켜지 않아도 자연스러운 진단 습관을 강제적으로 형성
② 이탈을 막는 습관을 설계한다 · 세부 기능

주간 맞춤 제품 및 리포트 발송

매주 맞춤 제품 배송 및 진단 리포트 발송
SOLUTION
  • 매주 리포트로 지난주 대비 내 피부 변화를 정기 확인 → 계속할 이유 제공
  • 리포트–제품이 세트로 도착 → 받은 제품을 써야 다음 리포트가 의미 있다는 사용 루프(락인) 형성
  • 진단을 빼먹으면 '빼먹을수록 리포트 정확도가 떨어져요' 안내 → 꾸준한 참여 유도
② 이탈을 막는 습관을 설계한다 · 세부 기능

포인트 · 환급 제도

출석·관리 활동으로 포인트를 적립하고 혜택·환급으로 전환
SOLUTION
  • 출석 및 피부관리를 통한 포인트 적립 행동별 포인트로 게이미피케이션
  • 이용자 내 순위 노출로 경쟁심 자극 · 매주 월요일 00:00 초기화
  • 적립 포인트를 혜택으로 전환
  • 장기 연속 출석 시 일정 금액 환급 제도 시행
② 이탈을 막는 습관을 설계한다 · 세부 기능

커뮤니티 서비스

비슷한 고민을 가진 사용자들과 채팅으로 소통하는 공간
SOLUTION
  • 비슷한 고민끼리 모이는 고민별 채팅방으로 정보·경험 공유 → 혼자 관리할 때보다 지속률↑
  • 리뷰 작성 시 포인트 지급으로 커뮤니티 참여와 락인을 연결

정밀 분석 시각자료

직관적인 부위별 / 항목별 시각 분석 자료 제공
SOLUTION
